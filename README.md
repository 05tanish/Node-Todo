# Node Todo App

> **Note:** This was my very first project! I built it when I was just starting out, but I never pushed it to GitHub until now. 🌱

A full-stack Todo application built with **Node.js**, **Express**, **MongoDB**, and **React**. It features user authentication, a responsive design, and full CRUD functionality for managing tasks.

## Features (Current & Upcoming)
- User Authentication (Login, Register, Profile, Change Password)
- Secure API with JWT
- Todo Management (Add, Edit, Delete, Toggle Complete) *(Coming in next commit)*

## Project Structure
- `frontend/`: React frontend powered by Vite and styled with Tailwind CSS.
- `backend/`: Node.js backend providing RESTful APIs.

## Getting Started
Navigate into the `frontend` or `backend` directories and check their respective README files for specific instructions on how to start the development servers.
