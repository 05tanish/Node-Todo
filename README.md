# Node Todo App

> **Note:** This was my very first project! Built when I was just starting out and finally pushed to GitHub. 🌱

A full-stack Todo application built with **Node.js**, **Express**, **MongoDB**, and **React**, featuring secure user authentication and full CRUD functionality.

## Core Features
- **User Authentication**: Login, Register, Profile, and Change Password secured with JWT.
- **Todo Management**: Complete CRUD operations (Add, Edit, Delete, Toggle status).
- **Responsive UI**: Styled beautifully with Tailwind CSS.

## Project Structure & Setup
- `frontend/`: React frontend powered by Vite.
- `backend/`: Node.js backend providing the RESTful API.

To get started, navigate into the `frontend` and `backend` directories and check their respective README files for local setup instructions.
